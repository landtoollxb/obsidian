# 总览速查（Ubuntu 24.04）

目标：一页快速查到“最常用命令 + 最短可复制示例”。需要更完整说明就跳到对应分类页。

## 0. 先记住的 3 个习惯
- 不确定时：先看帮助 `cmd --help` / `man cmd` / `tldr cmd`（可选安装）
- 改配置前：先备份 `sudo cp file file.bak.$(date +%F_%H%M)`
- 出问题先定位：`journalctl -u 服务名 -e` + `ss -lntp` + `df -h`

## 1. 文件与目录
- 当前目录：`pwd`
- 列出文件：`ls -alh`
- 进入目录：`cd /path`
- 创建目录：`mkdir -p /data/app/logs`
- 复制/移动：`cp -a src/ dst/`、`mv a b`
- 删除（谨慎）：`rm -rf /path`
- 查找文件：`find /etc -name "ssh*" -type f`
- 统计目录大小：`du -sh /var/log/* | sort -h`
- 查看磁盘占用：`df -hT`
- 查看树（可选）：`tree -L 2`（需要安装 tree）

## 2. 查看文件/日志
- 看前/后几行：`head -n 50 file`、`tail -n 200 file`
- 持续跟踪：`tail -f /var/log/syslog`
- 分页查看：`less -R file`（`/关键词` 搜索，`q` 退出）
- 查看文件类型：`file xxx`

## 3. 文本检索与处理
- 搜索关键词：`grep -RIn "keyword" /etc`
- 统计行数：`wc -l file`
- 去重计数：`sort file | uniq -c | sort -nr`
- 列处理（awk）：`awk '{print $1,$NF}' file`
- 替换（sed）：`sed -i 's/old/new/g' file`
- 管道组合：`cmd1 | cmd2 | cmd3`

## 4. 权限与用户
- 查看当前用户：`whoami`、`id`
- 文件权限：`ls -l`
- 修改权限：`chmod 644 file`、`chmod -R 755 dir`
- 修改属主：`sudo chown -R user:group /data/app`
- 切换用户：`sudo -iu user`
- 组与权限：`groups user`、`sudo usermod -aG sudo user`

## 5. 进程与服务（systemd）
- 看进程：`ps aux | grep name`
- 实时监控：`top` / `htop`（可选）
- 杀进程：`kill -15 PID`（优先）、`kill -9 PID`（最后手段）
- 服务状态：`systemctl status xxx -l`
- 启停与自启：`sudo systemctl start/stop/restart/enable/disable xxx`
- 看日志：`journalctl -u xxx -e`、`journalctl -u xxx -f`

## 6. 网络与端口
- IP/路由：`ip a`、`ip r`
- DNS：`resolvectl status`
- 监听端口：`ss -lntp`
- 连通性：`ping -c 4 1.1.1.1`
- HTTP 测试：`curl -I https://example.com`
- 下载：`wget URL -O file`
- 远程登录：`ssh user@host`

## 7. 传输与同步
- 拷贝到远端：`scp file user@host:/path/`
- 目录同步：`rsync -avP dir/ user@host:/path/dir/`
- 免密（部署常用）：`ssh-keygen` + `ssh-copy-id user@host`

## 8. 包管理（Ubuntu 24.04）
- 更新索引：`sudo apt update`
- 安装软件：`sudo apt install -y pkg`
- 查包：`apt-cache search keyword`
- 卸载：`sudo apt remove pkg`、`sudo apt purge pkg`
- 查看已装：`apt list --installed | grep pkg`
- snap：`snap list`、`sudo snap install xxx`

## 9. 磁盘与挂载（部署前置）
- 看磁盘：`lsblk -f`
- 分区信息：`sudo fdisk -l`
- 挂载：`sudo mount /dev/sdb1 /data`
- 持久挂载：编辑 `/etc/fstab`（谨慎，改错会影响启动）
- 权限：`sudo chown -R user:group /data`

## 10. 性能与排障
- CPU/内存：`top`、`free -h`
- 负载：`uptime`
- 进程占用：`ps -eo pid,cmd,%cpu,%mem --sort=-%cpu | head`
- 打开文件数：`lsof -p PID | wc -l`
- 端口占用：`sudo lsof -i :8080`
- 系统信息：`uname -a`、`lsb_release -a`
