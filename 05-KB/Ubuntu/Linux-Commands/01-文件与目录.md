# 文件与目录

## 常用命令

### 路径与导航
```bash
pwd
cd /etc
cd -
```

### 列表与查看
```bash
ls -alh
ls -alh --time-style=long-iso
stat file
file file
```

### 创建/复制/移动/删除
```bash
mkdir -p /data/app/{bin,conf,logs}
cp -a src/ dst/
cp -av file{,.bak}
mv old new
rm -i file
rm -rf dir
```

### 查找（find）
```bash
find /etc -type f -name "*.conf"
find /data -maxdepth 2 -type d -name "logs"
find . -type f -mtime -7
find . -type f -size +100M
```

### 快速定位（locate）
```bash
sudo apt install -y plocate
sudo updatedb
locate sshd_config
```

### 软链接/硬链接
```bash
ln -s /opt/java/current /usr/local/java
ln file hardlink_file
readlink -f /usr/local/java
```

### 压缩/解压
```bash
tar -czf backup.tgz dir/
tar -xzf backup.tgz
zip -r backup.zip dir/
unzip backup.zip
```

### 权限与属主（基础）
```bash
chmod 644 file
chmod -R 755 dir
sudo chown -R user:group /data/app
```

## 实战片段

### 1) 找到“谁占了最多空间”
```bash
df -hT
sudo du -xhd1 / | sort -h
sudo du -xhd1 /var | sort -h
```

### 2) 批量备份配置文件
```bash
sudo cp /etc/ssh/sshd_config /etc/ssh/sshd_config.bak.$(date +%F_%H%M)
```

### 3) 统计某目录下文件数量
```bash
find /data/app -type f | wc -l
```

## 常见坑
- `rm -rf` 不可撤销，先用 `ls`/`find` 确认目标路径。
- 用 `cp -a` 复制目录更稳（保留权限、时间戳、软链接）。
- 大数据部署常用数据盘建议挂载到独立目录（如 `/data`），并确保属主/权限正确。
