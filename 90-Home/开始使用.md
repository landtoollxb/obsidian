# 开始使用

## 1. 打开 Vault
- Obsidian → Open folder as vault → 选择 `E:\obsidian\ObsidianVault`

## 2. 建议安装的插件（先装这些）
- Obsidian Git
- Templater
- Calendar
- Dataview
- QuickAdd

## 3. 推荐目录结构
- 00-Inbox：临时捕获/待整理
- 01-Daily：日记
- 03-Reading：读书
- 04-DevNotes：代码笔记
- 05-KB：知识库条目
- 90-Home：导航与索引
- 99-Templates：模板（给 Templater/QuickAdd 使用）

## 4. 插件基础配置（按这个填）
### 4.1 Templater
- Template folder location：`99-Templates`

### 4.2 Daily Notes（内置核心插件）
- Folder：`01-Daily`
- Date format：`YYYY-MM-DD`
- Template file location：`99-Templates/Daily.md`

### 4.3 Calendar
- 打开 Calendar 侧边栏即可使用（与 Daily Notes 配合）

### 4.4 Dataview
- 安装后打开 [主页.md](file:///E:/obsidian/ObsidianVault/90-Home/%E4%B8%BB%E9%A1%B5.md)

### 4.5 QuickAdd
- 安装后先手动创建 3 个 Capture（后面有建议配置）：日记/读书/代码笔记

## 5. Git 同步（电脑端）
### 5.1 远端已配置
- 仓库：`git@github.com:landtoollxb/obsidian.git`

### 5.2 电脑端同步方式建议
- 编辑前先 Pull
- 结束编辑后 Commit-and-sync
